# Cloud-Automation-Chatbot

3개의 에이전트를 이용한 클라우드 인프라 설계 챗봇

주요 기능 

Multi-Agent Workflow: 설계(YAML), 개발(Terraform), 검증(Validator) 에이전트 간의 유기적인 협업.

Self-healing IaC: Terraform 실행 로그를 분석하여 AI가 스스로 코드를 수정하는 자율 교정 기능.

RAG 기반 지식 베이스: AWS Well-Architected Framework 데이터를 활용한 고신뢰도 아키텍처 추천.

실시간 스트리밍 (SSE): AI의 사고 과정과 인프라 실행 로그를 실시간으로 중계하는 투명한 UI.

비용 분석 및 시각화: Infracost 연동을 통한 예상 비용 산출 및 diagrams-as-code 기반 다이어그램 생성.
